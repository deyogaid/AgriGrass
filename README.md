# AgriGrass (tanicuan)
visi produk
Aplikasi transparansi tani


Petani terjerat kemiskinan sistemik karena rantai distribusi yang panjang (5-8 titik) dan ketergantungan pada sistem "Ijon" untuk modal cepat, sementara mereka buta terhadap harga pasar real-time.  
Tujuan Platform: Menciptakan "Single Source of Truth" yang mendemokrasikan data harga pasar induk dan menyediakan prediksi harga berbasis AI agar petani memiliki daya tawar tinggi.
